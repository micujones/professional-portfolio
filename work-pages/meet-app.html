Meet App

The Meet App is a single-page, progressive web application that displays events in various cities. Users can filter the list based on the available cities, choose a number of events to be displayed, and observe data visualization of the events rendered.

Features

  • Filter events by city
  • Show and hide event details
  • Specify the number of events shown
  • Use the app offline
  • Add a shortcut to desktop and mobile home screens
  • Display visualization data of details

Tech Stack

  • AWS Lambda
  • Google Calendar API
  • OAuth2 Authentication
  • React.js
  • Recharts
  • Vercel
